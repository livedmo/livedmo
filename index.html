Good Homolosine Map

This map is from Lab Report #2. I was tasked with correcting projections and providing a description of three new projections of my choice. I chose the Good Homolosine map, which is particularly good for data collection. However, this map is made up of spheres, so, some of the projections are distorted. To create this map, I had to use on-the-fly projections, change the coordinate system, and set a new projection.

Population Density Per Tract Map

This map is from Mastery Assignment #2. I was asked to create a Choropleth map showing the population density. I was allowed to choose which upper right-hand section of the map that I wanted. To create this map I had to look at the attributable table, use symbology to create the choropleth, and normalize the data.

Philadelphia Schools Map

This map came from Lab Report #1. For this assignment, I had to identify schools in Philadelphia that have more than 1,200 students and within 0.1 mile of a bike network. To create this map I used select by attribute, select by location, and the symbology tab.
